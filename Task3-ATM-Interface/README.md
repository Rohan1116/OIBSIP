# ATM Interface System

## Overview

This project is a console-based ATM Interface System developed using Java.
The application is designed to perform basic banking operations while demonstrating core Java programming and Object Oriented Programming concepts.

The system allows users to securely log in using a PIN and access different ATM functionalities such as balance checking, depositing money, withdrawing money, transferring funds, and viewing transaction history.

This project was developed as part of the Oasis Infobyte Java Development Internship.

---

## Features

- PIN Based Login System
- Check Account Balance
- Deposit Money
- Withdraw Money
- Transfer Funds
- Transaction History
- Menu Driven Program
- Simple and User-Friendly Interface

---

## Technologies Used

- Java
- OOP Concepts
- ArrayList
- Scanner Class
